# Ruth — Referral SOP

**Owner:** Ruth (Client Success)
**Goal:** 2 referral asks per day, 5 referral leads per week
**Source:** Passion's spreadsheet — [Referral tab](https://docs.google.com/spreadsheets/d/1asUjGTbEyhhHaqCWNZsP-GXioYh3xE9O/edit?gid=1062390516#gid=1062390516)

## Tracking System

### Referral Tracking Database
Every referral ask Ruth makes gets logged here.

| Field | Description |
|---|---|
| Date | Date the ask was made |
| Client Name | Existing Jarvis client who was asked |
| Trigger Type | What prompted the ask (see triggers below) |
| Asked? (Y/N) | Did Ruth actually ask for the referral |
| Script Sent? | Was the referral script sent to the client |
| Lead Name | Name of the referred lead (if captured) |

### Top Referrers Leaderboard
Tracks which clients refer the most — used for VIP recognition.

| Field | Description |
|---|---|
| Client Name | Jarvis client |
| # Referrals Given | Total referrals they've sent |
| # Closed | How many of their referrals converted to clients |
| Revenue Generated | Revenue from their referrals |
| Last Referral Date | Most recent referral date |
| VIP Status | Recognition tier for top referrers |

---

## Trigger Types (when to ask)

Ruth should ask for a referral when any of these moments happen:

- **Onboarding complete** — client just had their VA set up and is excited
- **Positive check-in** — client expresses satisfaction during a weekly check-in
- **Problem resolved** — Ruth solved an issue, client is relieved/grateful
- **Milestone hit** — client reports a win (saved time, grew revenue, etc.)
- **Upsell accepted** — client just added a second VA (high trust moment)

---

## Daily Targets

- 2 referral asks per day
- Script sent with every ask
- Lead name captured and logged same day
- 5 referral leads per week feeding into GHL pipeline
