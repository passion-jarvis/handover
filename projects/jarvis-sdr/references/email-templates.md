## GHL Workflow Setup Notes

**Trigger:** Contact tag = `job-signal`

**Flow:**
1. Wait 2 hours → send Email 1 (match template to `cf_keyword_category` tag)
2. Wait 3 days → check if opportunity stage moved → if not, send Email 2
3. Wait 3 days → check again → if not moved, send Email 3
4. Wait 1 day → if still no movement → create Task for SDR: "Manual LinkedIn outreach — {{contact.first_name}} at {{contact.company_name}}"

**Sender:** passion@gojarvis.ai (or whichever email is connected to GHL)
**Reply-to:** same

**Personalization check before activating:**
- Confirm `cf_keyword_category` is populated on all contacts entering this workflow
- Add Calendly link to all Email 1 templates
- Test with a real contact on `DRY_RUN` mode before turning on automations
